# BioDesign-AI
BioDesignAI is an AI-powered engineering application for designing biogas plants through automated numerical analysis, engineering visualization, decision support, and comprehensive design report generation.

# 🌱 BioDesignAI

AI-Powered Biogas Plant Design & Decision Support System

## Features

- Engineering Calculations
- Fixed Dome Biogas Design
- Engineering Drawing
- 3D Visualization
- Cost Estimation
- PDF Report Generation

## Tech Stack

- Python
- Streamlit
- Plotly
- Matplotlib
- Pandas
- NumPy
